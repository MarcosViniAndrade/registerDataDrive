## Welcome to RegisterDataDrive

This project is a http route written in Node.js. The proposal of this route is register a driver and a follow up on DataDrive service.

## Getting Started

1. Clone the project:

```
git clone git@github.com:MarcosViniAndrade/registerDataDrive.git
```

2. Check `.env.sample` and set `CREDENTIALS` constant in a .env file.

## Usage

##### Run server

```
make run
```

##### Test request

```
make test
```
